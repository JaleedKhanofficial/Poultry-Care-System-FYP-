Poultry Care System

A comprehensive web application designed to enhance poultry farm management by automating processes, improving inventory and incubation record tracking, and ensuring real-time data access for decision-making. Developed for Marhaba Poultry Farm, the system aims to boost operational efficiency, streamline workflows, and support industry compliance with features like automated inventory alerts, detailed reporting, and secure data management. Built with React JS for an intuitive frontend experience and MySQL with REST APIs for seamless backend functionality, the project addresses common challenges in hatchery management, from inventory to sales and health monitoring.

Technologies Used: React JS, Bootstrap, MySQL, REST APIs.
Key Features: Inventory and stock management, incubation tracking, sales and expense management, role-based access control, and extensive reporting.
